# GDQuest Godot Shaders

Large MIT library of 2D/3D Godot shaders with demos (from GDQuest Shader Secrets / demos repo). **Shader/code = MIT (commercial OK).** Bundled **art textures/models = CC-BY-NC-SA 4.0** — do not ship those art files in commercial builds; swap in your own or CC0 art.

## Notes

- Godot 4 port in progress; Godot 3 tagged release `1.0.0` if you need a known-good 3.x tree
- Prefer copying individual `.gdshader` files; leave NC demo art behind
- Attribution example (shaders): `GDQuest Godot Shaders © GDQuest — MIT`

## Evidence

- Live `LICENSE` (2026-07-19): “demo's source code (script files, scene files, shaders…) … MIT”; “Art assets (image textures and 3D models) are CC-BY-NC-SA 4.0.”
