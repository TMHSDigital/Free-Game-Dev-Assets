# Godot Shaders

Community canvas-item / spatial shader library for Godot. Site policy allows personal **and commercial** use of posted shader **code**, but each post picks **CC0, MIT, or GPL v3** — GPL can infect closed-source builds. Filter per page; prefer CC0/MIT.

## Notes

- License page lists only CC0 / MIT / GPL v3 for submitted shader code
- Featured images, screenshots, videos, and embedded assets are **not** covered by the shader license
- Do not promote a shader to `active` without reading that post’s license badge
- GPL shaders → treat as high-risk for proprietary games

## Evidence

- Live [License](https://godotshaders.com/license/) (2026-07-19): “Anyone will be able to use the shaders posted to the site freely, both in personal and commercial products.” Submitters choose CC0, MIT, or GNU GPL v.3. Exception: media/assets in posts are outside that license.
