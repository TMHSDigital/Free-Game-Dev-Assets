# Noto Sans

Google’s SIL OFL family with broad script coverage including CJK — default pick for localization-friendly UI text. Bundle the faces you need; do not treat “download free” as a separate license.

## Notes

- License source of truth: Noto project `LICENSE` (SIL OFL 1.1) — also mirrored via Google Fonts
- OFL: embed/bundle/sell **with** software OK; cannot sell the font files alone; derivatives stay OFL
- Keep OFL notice with redistributed font binaries
- Attribution example: `Noto Sans © The Noto Project Authors — SIL OFL 1.1`

## Evidence

- Live-checked `https://raw.githubusercontent.com/notofonts/noto-fonts/main/LICENSE` (2026-07-19): “This Font Software is licensed under the SIL Open Font License, Version 1.1.” — OFL text states fonts may be “bundled, embedded, redistributed and/or sold with any software.”
- Google Fonts Noto hub (`fonts.google.com/noto`) remains the distribution entrypoint; page is JS-heavy so license verified via upstream LICENSE file.
