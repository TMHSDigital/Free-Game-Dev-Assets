# Sim Daltonism

Apache-2.0 color-blindness simulator (Mac screen filter + iOS camera). Design/playtest tool — not shipped in-game. Complements [ubisoft-chroma](ubisoft-chroma.md) / [daltonlens](daltonlens.md).

## Notes

- Open source: github.com/michelf/sim-daltonism (`Apache License 2.0.txt`)
- Apple platforms only; use DaltonLens/Chroma on Windows
