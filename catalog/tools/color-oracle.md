# Color Oracle

Free full-screen colorblindness simulator (Windows/Mac/Linux) for deuteranopia/protanopia/tritanopia checks. Design-time a11y tool, not an in-game filter.

## Notes

- Site points to GitHub: [color-oracle](https://github.com/nvkelso/color-oracle) (Mac) and [color-oracle-java](https://github.com/nvkelso/color-oracle-java) (Win/Linux).
- Complements [sim-daltonism](sim-daltonism.md) / [daltonlens](daltonlens.md).
