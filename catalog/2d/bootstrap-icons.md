# Bootstrap Icons

2000+ MIT SVG/web-font icons — denser fill/outline set than Lucide/Heroicons for menus, settings, and HUD chrome.

## Notes

- Prefer [lucide-icons](lucide-icons.md) / [phosphor-icons](phosphor-icons.md) when you want thinner stroke systems; this fills the Bootstrap-shaped gap
- Keep MIT copyright notice when redistributing the library sources
