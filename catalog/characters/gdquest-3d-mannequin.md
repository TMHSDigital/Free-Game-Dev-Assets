# GDQuest Open 3D Mannequin

Open mannequin mesh plus a third-person controller sample for Godot. Good placeholder humanoid while art is unfinished; art is CC-BY-4.0 (code MIT).

## Notes

- Repo targets **Godot 3.2+** controller scenes — treat the mesh/animations as the reusable catalog value; expect porting work for Godot 4.
- Attribute: `GDQuest and contributors (https://www.gdquest.com/)`.
- Not a character generator — one stylized mannequin, not a wardrobe system.

## Evidence

- Root `LICENSE`: dual license — source MIT; art assets CC-BY-4.0 with attribution to GDQuest and contributors.
- README restates the same split.
