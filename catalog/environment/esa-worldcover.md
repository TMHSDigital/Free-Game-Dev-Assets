# ESA WorldCover

10 m global land-cover maps (2020/2021) + annual Sentinel composites. Good for biome masks, foliage density, and procedural biome painting — not a heightmap.

## Notes

- Free via Terrascope / AWS Open Data. Attribute per CC-BY-4.0 and cite the dataset/paper as on the data-access page.
- 2020 vs 2021 use different algorithm versions — don’t treat year-to-year diffs as pure real change.
