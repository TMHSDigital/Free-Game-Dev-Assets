# Microsoft Global Building Footprints

~1.4B ML-detected building footprints (many with height estimates) from Bing Maps imagery. Useful for city kits, proxy LOD0 buildings, and OSM gap-fill — not as a ToS-safe Google/Bing mesh scrape.

## Notes

- Redistributing the **Data** requires including CDLA Permissive 2.0 text. Derived “Results” (e.g. analysis outputs) are unrestricted under the license.
- Imagery sources vary by region; treat footprints as approximate.

- README: “freely available for download and use under CDLA Permissive 2.0.”
