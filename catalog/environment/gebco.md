# GEBCO Grid

Global ocean + land terrain model (~15″). Best free bathymetry source for undersea levels, coastal games, and planet-scale heightmaps. **Not for navigation.**

## Notes

- Current release: GEBCO_2026 (tiles / global / OPeNDAP).
- Must attribute GEBCO; must not imply IHO/IOC endorsement.
- Related land DEMs: [nasadem](nasadem.md), [etopo-2022](etopo-2022.md), [jaxa-alos-aw3d30](jaxa-alos-aw3d30.md).
