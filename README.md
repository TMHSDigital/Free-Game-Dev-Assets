# Free Game Dev Assets

Public catalog of **free / commercially usable** assets and tools for game development (and related interactive work).

This repo **links and documents** sources. It does **not** rehost third-party binary packs.

## Browse

Full index: [`catalog/`](catalog/) · License primer: [`docs/licenses.md`](docs/licenses.md)

## How entries work

Each source is a markdown file with YAML frontmatter (`license`, `commercial`, `attribution_required`, `verified`, `status`, …). Copy [`catalog/TEMPLATE.md`](catalog/TEMPLATE.md) to add one.

`status: needs-review` means useful discovery, not a green light — re-check the live license before shipping a game.

## Contributing

See [`CONTRIBUTING.md`](CONTRIBUTING.md). PRs welcome for new sources, license corrections, and dead-link fixes.

## Scope

- Prefer CC0 / public domain / SIL OFL / clearly commercial-ok custom licenses
- Aggregators (OpenGameArt, itch collections, Poly Pizza) are listed with `varies` — filter per item
- Engine-store “free” assets stay as outbound links only (no redistribution)
